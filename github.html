Curso GIT da Alura

Github - Trabalhando com remotes

  • git remote add apelido-do-remoto url (para adicionar a url remoto)
  • git push (para empurrar/uploadar os commits para remoto)
  • git pull eh apelido para git fetch e git merge origin/master
  • git pull (para pegar os commits do remoto)
  • git remote -v (para listar todos os remotos cadastrados)
  • git status (para ver se estamos em sync com remote)
